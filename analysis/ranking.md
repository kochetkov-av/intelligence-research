# Стратегические приоритеты

## Если можете делать только 3 вещи:

**🛡️ Поддержание:**
1. Сон 7-8 часов + качество (83.8 / 75.0)
2. Движение 150+ мин/нед (75.6)
3. Социальные связи (73.8)

**🚀 Развитие:**
1. Обучение других (75.3)
2. Решение сложных задач (74.1)
3. HIIT (72.8)

## Формула:

```
🛡️ ПОДДЕРЖАНИЕ:
   Сон (количество + качество) + Движение + Социум + Питание (MIND)
   - Алкоголь - UPF - Стресс

🚀 РАЗВИТИЕ (после стабилизации базы):
   Обучение других + Сложные задачи + HIIT + Творчество + Медитация
```
