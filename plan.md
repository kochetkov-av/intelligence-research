# План исследования: Факторы развития и поддержания интеллекта

## Цель исследования
Создать комплексный обзор факторов, влияющих на когнитивные способности взрослых (25-65 лет), с научным обоснованием и практическими рекомендациями. Итог — структурированный отчёт со скорингом важности факторов.

---

## Фаза 1: Теоретическая база

### 1.1 Определения и концепции интеллекта
- [ ] g-фактор (общий интеллект) — Спирмен, современные исследования
- [ ] Флюидный vs кристаллизованный интеллект (Gf/Gc) — Кеттелл
- [ ] Множественные интеллекты — Гарднер (8 типов)
- [ ] Триархическая теория — Стернберг (аналитический, творческий, практический)
- [ ] Эмоциональный интеллект — Гоулман, Майер-Саловей
- [ ] Рабочая память как ключевой компонент

### 1.2 Нейробиологические основы
- [ ] Нейропластичность во взрослом возрасте
- [ ] Нейрогенез (гиппокамп)
- [ ] Миелинизация и скорость обработки
- [ ] Синаптическая пластичность
- [ ] Роль нейротрансмиттеров (дофамин, ацетилхолин, BDNF)

  - Качество vs количество

- [ ] **Физическая активность**

- [ ] **Питание**

- [ ] **Обучение и образование**

### 2.8 Ноотропы и нейропротекторы (полный обзор)

## Фаза 5: Система скоринга

### 5.1 Параметры оценки каждого фактора

| Параметр | Описание | Шкала |
|----------|----------|-------|
| **Сила эффекта** | Насколько сильно влияет на интеллект | 1-10 |
| **Доказательная база** | Качество научных данных | 1-10 |
| **Модифицируемость** | Можно ли изменить фактор самостоятельно | 1-10 |
| **Доступность** | Легкость внедрения | 1-10 |
| **Скорость эффекта** | Как быстро заметен результат | 1-10 |
| **Устойчивость** | Долгосрочность эффекта | 1-10 |

### 5.2 Итоговый скоринг
- Взвешенная оценка по параметрам
- Ранжирование факторов
- Категории: критические / важные / желательные / опциональные
