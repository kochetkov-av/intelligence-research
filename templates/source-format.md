# Формат источников и цитирования

## 1. Общие принципы

- Все утверждения о факторах должны подкрепляться ссылками
- Приоритет отдаётся источникам с высоким уровнем доказательности
- При противоречиях между источниками указывать оба с комментарием
- Предпочтение отдаётся рецензируемым журналам

---

## 2. Иерархия уровней доказательности

### Уровень A — Высший уровень доказательности

| Подуровень | Тип исследования | Описание |
|------------|------------------|----------|
| A1 | Кокрейновский систематический обзор | Золотой стандарт доказательной медицины |
| A2 | Мета-анализ высокого качества | ≥10 RCT, низкая гетерогенность (I² < 50%), строгие критерии включения |
| A3 | Мета-анализ среднего качества | 5-10 RCT, умеренная гетерогенность |

### Уровень B — Высокий уровень доказательности

| Подуровень | Тип исследования | Описание |
|------------|------------------|----------|
| B1 | RCT высокого качества | N > 100, двойное слепое, плацебо-контроль, низкий риск смещения |
| B2 | RCT среднего качества | N > 50, адекватная рандомизация и ослепление |
| B3 | Множественные RCT | Несколько RCT с согласованными результатами |

### Уровень C — Средний уровень доказательности

| Подуровень | Тип исследования | Описание |
|------------|------------------|----------|
| C1 | RCT низкого качества | Малая выборка, проблемы с ослеплением или рандомизацией |
| C2 | Проспективное когортное | Крупное когортное исследование с длительным follow-up |
| C3 | Контролируемое без рандомизации | Quasi-experimental design |

### Уровень D — Низкий уровень доказательности

| Подуровень | Тип исследования | Описание |
|------------|------------------|----------|
| D1 | Ретроспективное когортное | Case-control, ретроспективный анализ |
| D2 | Кросс-секционное | Одномоментное обсервационное исследование |
| D3 | Case series | Серия случаев (N > 10) |

### Уровень E — Минимальный уровень доказательности

| Подуровень | Тип исследования | Описание |
|------------|------------------|----------|
| E1 | Case report | Единичные случаи |
| E2 | Экспертное мнение | Консенсус экспертов, клинические рекомендации без эмпирической базы |
| E3 | Теоретическая модель | Экстраполяции, in vitro, животные модели |

## 3. Шаблон цитирования

### 3.1 Стандартный формат ссылки

```
[Номер] Автор(ы) (Год). Название статьи. Журнал, Том(Номер), страницы. DOI. [Уровень доказательности]
```

### 3.2 Примеры оформления

#### Мета-анализ
```
[1] Smith J, Jones K, Brown L. (2023). Effects of aerobic exercise on cognitive function: A systematic review and meta-analysis. Neuroscience & Biobehavioral Reviews, 142, 104892. doi:10.1016/j.neubiorev.2023.104892 [A2]
```

#### RCT
```
[2] Иванов А.А., Петров Б.Б. (2022). Влияние омега-3 на рабочую память: рандомизированное плацебо-контролируемое исследование. Журнал неврологии, 45(3), 234-248. doi:10.xxxx/xxxxx [B1]
```

#### Когортное исследование
```
[3] Wilson M et al. (2021). Long-term cognitive outcomes associated with Mediterranean diet adherence: The PREDIMED study. JAMA Neurology, 78(4), 421-430. doi:10.1001/jamaneurol.2021.0012 [C2]
```

### 3.3 Сокращённый формат (для таблиц)

```
Smith et al., 2023 [A2]
Иванов и Петров, 2022 [B1]
```

## 4. Правила указания авторов

| Количество авторов | Формат |
|-------------------|--------|
| 1 автор | Фамилия И.О. |
| 2 автора | Фамилия И.О., Фамилия И.О. |
| 3-6 авторов | Все авторы при первом упоминании, затем "и др." / "et al." |
| > 6 авторов | Первые 3 автора + "и др." / "et al." |

## 5. Специальные обозначения

### 5.1 Качество исследования

| Обозначение | Значение |
|-------------|----------|
| ★★★ | Высокое качество (низкий риск смещения) |
| ★★☆ | Среднее качество (некоторые ограничения) |
| ★☆☆ | Низкое качество (существенные методологические проблемы) |

### 5.2 Конфликт интересов

| Обозначение | Значение |
|-------------|----------|
| [COI-] | Конфликт интересов не заявлен |
| [COI+] | Заявлен конфликт интересов (требует осторожной интерпретации) |
| [IND] | Финансирование от индустрии |

### 5.3 Воспроизводимость

| Обозначение | Значение |
|-------------|----------|
| [REP+] | Результаты воспроизведены в независимых исследованиях |
| [REP-] | Попытки репликации не удались |
| [REP?] | Репликации не проводились |

## 8. Рекомендации по работе с источниками

### 8.1 Поиск источников

1. **PubMed/MEDLINE** — основная база для медицинских исследований
2. **Cochrane Library** — систематические обзоры
3. **Google Scholar** — широкий охват, включая препринты
4. **PsycINFO** — психологические исследования
5. **Semantic Scholar** — AI-powered поиск

### 8.2 Оценка качества

При оценке качества исследования учитывать:
- Размер выборки (N)
- Методы рандомизации и ослепления
- Контрольная группа
- Длительность follow-up
- Первичные vs вторичные исходы
- Риск смещения (bias)
- Конфликт интересов

### 8.3 Работа с противоречиями

Когда исследования показывают противоречивые результаты:
1. Указать оба источника
2. Проанализировать возможные причины расхождений
3. Отдать приоритет исследованиям более высокого уровня
4. Отметить неопределённость в выводах

## 9. Пример полного оформления

```markdown
### Источники

#### Основные источники

1. Northey JM, Cherbuin N, Pumpa KL et al. (2018). Exercise interventions for cognitive function in adults older than 50: a systematic review with meta-analysis. British Journal of Sports Medicine, 52(3), 154-160. doi:10.1136/bjsports-2016-096587 [A2] ★★★ [COI-] [REP+]

2. Erickson KI, Voss MW, Prakash RS et al. (2011). Exercise training increases size of hippocampus and improves memory. Proceedings of the National Academy of Sciences, 108(7), 3017-3022. doi:10.1073/pnas.1015950108 [B1] ★★★ [COI-] [REP+]

#### Поддерживающие источники

3. Colcombe S, Kramer AF. (2003). Fitness effects on the cognitive function of older adults: a meta-analytic study. Psychological Science, 14(2), 125-130. doi:10.1111/1467-9280.t01-1-01430 [A3] ★★☆ [COI-] [REP+]

#### Противоречащие источники

4. Young J, Angevaren M, Rusted J, Tabet N. (2015). Aerobic exercise to improve cognitive function in older people without known cognitive impairment. Cochrane Database of Systematic Reviews, 4, CD005381. doi:10.1002/14651858.CD005381.pub4 [A1] ★★★ [COI-]
   *Примечание: Кокрейновский обзор показал более скромные эффекты, чем предыдущие мета-анализы*

---
Дата проверки актуальности источников: 2024-01-15
```
